# Security Policy

Koschei Web3 Hub is a no-custody Web3 intelligence project. Responsible disclosure helps keep users and builders safer.

## Responsible Disclosure

If you discover a security issue, please report it privately if possible before sharing details publicly.

Contact: onur24sel@gmail.com

Please include enough detail to help reproduce and understand the issue, but do not include unnecessary sensitive information.

## Do Not Share Sensitive Material

Never share private keys or seed phrases with Koschei, contributors or maintainers.

Do not include any of the following in reports, issues, pull requests or screenshots:

- Private keys.
- Seed phrases.
- Passwords.
- Session tokens.
- API keys.
- Sensitive user data.

## Public Disclosure

Please do not publish exploit details publicly before a fix is available or before maintainers have had a reasonable opportunity to investigate.

Security reports should focus on responsible remediation and user safety.
