# Koschei ARVIS Grant V2 Proposal Draft

## Project name

Koschei ARVIS — Solana Security Infrastructure

## One-line summary

Koschei ARVIS is a Solana-native risk intelligence infrastructure layer for launch, liquidity, wallet and claim risk, designed for builders, wallets, launchpads and security teams.

## Problem

Solana users and builders face fast-moving risk around new tokens, liquidity pools, suspicious wallets and claim surfaces. Existing workflows often require manual checks across multiple tools. Builders need a reusable infrastructure layer that converts live Solana observations into evidence-backed verdicts.

## Solution

Koschei ARVIS collects live Solana observations, normalizes targets, runs evidence checks, and produces signed verdicts through radar, reports and API surfaces.

The project is now positioned as technical infrastructure, not a community-only hub.

## Technical deliverables

1. Live Solana radar infrastructure
2. Signed verdict schema
3. Developer API documentation
4. Open-source event normalization helpers
5. SDK starter and integration examples
6. Architecture and production proof documentation
