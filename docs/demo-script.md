# Koschei ARVIS Demo Script

## Goal

Show that Koschei ARVIS is a working Solana risk intelligence system, not a community-only project.

## 60 second demo

### 0-10 seconds: positioning

Koschei ARVIS is Solana security infrastructure for launch, liquidity, wallet and claim risk.

### 10-25 seconds: live radar

Open the live radar and show:

- source freshness
- completed jobs
- signed verdicts
- backlog and failure counters
- verified risk and monitor cards

### 25-40 seconds: verdict proof

Open one verified card and show:

- target
- grade
- risk index
- evidence list
- rule version
- signed status

### 40-50 seconds: developer resources

Open the developer page and show:

- API endpoints
- signed verdict schema
- open-source component roadmap

### 50-60 seconds: architecture

Open the architecture page and show the four layers:

1. source collection
2. evidence processing
3. signed verdict generation
4. developer delivery

## Closing line

Koschei ARVIS turns live Solana data into evidence-backed signed verdicts for builders, wallets, launchpads and security teams.
