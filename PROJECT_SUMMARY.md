# Project Summary

Koschei is a locked-scope MVP focused on:
- Web Game Factory
- Web3 Bridge readiness packaging

## What it does
- Converts prompts into playable HTML5 previews.
- Produces Web3-ready package outputs (manifest, metadata, rewards, adapter config).
- Persists projects and generated outputs in Neon Postgres.

## What it does not do
- No wallet custody
- No private key handling
- No minting
- No contract deployment
- No transaction signing

## Infrastructure
- Next.js + React
- Neon Postgres
- Railway
- Optional read-only/future Alchemy RPC alignment
