Developer resources

Solana risk intelligence API for builders.

Koschei ARVIS exposes a signed verdict pipeline that Solana wallets, launchpads, research teams and security products can consume. The goal is a developer-facing security primitive: live Solana data, verified evidence, deterministic risk output.

Open live radarView architecture
Endpoint

Risk check

General Solana target risk check for tokens, pools, wallets, programs, transactions or claim URLs.

POST /api/v1/risk/check { "target": "SOLANA_TARGET", "network": "solana-mainnet" }
Endpoint

Token scan

Token-focused scan for launch behavior, authority state, liquidity evidence and holder concentration.

POST /api/v1/token/scan { "mint": "TOKEN_MINT", "include_evidence": true }
Endpoint

Wallet analyze

Wallet intelligence surface for age, activity, counterparty risk and sybil hints.

POST /api/v1/wallet/analyze { "wallet": "SOLANA_WALLET" }
Endpoint

Radar feed

Live feed for verified cards, stream health and signed verdict production proof.

GET /api/v1/radar/feed
Schema

Signed verdict contract

Every final customer verdict should carry a stable structure that downstream tools can verify and display.

{ "grade": "A-F", "risk_index": 0, "risk_level": "low|medium|high|critical", "evidence": [], "rule_version": "arvis-live", "signed": true }
