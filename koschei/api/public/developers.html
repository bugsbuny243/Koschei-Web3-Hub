İmzadan önce tek, kanıta dayalı karar.

Cüzdanlar, dApp'ler, launchpad'ler ve güvenlik ekipleri; ayrı token, işlem, cüzdan ve alarm sistemleri kurmamak için ARVIS'i entegre eder. Çıktı makine tarafından okunabilir: kanıt ve kural bilgileriyle birlikte izin ver, uyar, engelle veya beklet.

Token-2022 Güvenlik Tarayıcısı

Klasik SPL Token ile Token-2022'yi ayırır; transfer, yetki, gizlilik ve uyumluluk davranışını değiştiren uzantıları değerlendirir.

POST /api/token/scan Authorization: Bearer <oturum-tokeni> { "mint": "TOKEN_MINT", "network": "solana-mainnet" }
