Copyable SDK Examples

No npm package is required. Use standard HTTP clients.

JavaScript fetch

const response = await fetch('/api/public/impact');
const impact = await response.json();

Python requests

import requests
impact = requests.get('https://your-host/api/public/impact').json()
print(impact)

curl

curl https://your-host/api/public/impact

Operation note

Webhook setup is not required. Watchlist uses Alchemy API polling.

No wallet connection or private keys required for core tools.
