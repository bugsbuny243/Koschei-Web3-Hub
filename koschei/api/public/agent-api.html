Koschei Agent API

Sınırlandırılmış, salt okunur Koschei iş akışları için geliştirici ve ajan araçları.

Temel araçlar için cüzdan bağlantısı veya özel anahtar gerekmez.

Kullanılabilir uç noktalar

GET /api/agent/healthPOST /api/agent/wallet-scorePOST /api/agent/risk-summaryPOST /api/agent/metadata-templatePOST /api/agent/chain-health

API anahtarı yönetimi

API anahtarları mevcut yönetici tanımlama akışı üzerinden yönetilir.

Yönetici anahtarları burada hiçbir zaman gösterilmez. Agent API çağrıları loglanır; MVP aşamasında çıktı hakkı tüketmez.

Örnek kullanım

curl -H "x-koschei-agent-key: ANAHTARINIZ" https://alan-adiniz/api/agent/health
