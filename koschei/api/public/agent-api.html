Koschei Agent API

Developer and agent tooling for scoped, read-only Koschei workflows.

No wallet connection or private keys required for core tools.

Available endpoints

GET /api/agent/healthPOST /api/agent/wallet-scorePOST /api/agent/risk-summaryPOST /api/agent/metadata-templatePOST /api/agent/chain-health

API key management

API key management is handled through the current administrator provisioning workflow.

Admin keys are never exposed here. Agent API calls are logged; outputs are not charged for the MVP.

Example

curl -H "x-koschei-agent-key: YOUR_KEY" https://your-host/api/agent/health
