Webhook setup guide
Webhook URL
https://tradepigloball.co/api/web3/events/alchemy
  1. Create an Address Activity webhook in Alchemy.
  2. Use the webhook URL above.
  3. Add addresses from your Koschei Watchlist.
  4. Copy the Alchemy signing key into Railway env as ALCHEMY_WEBHOOK_SIGNING_KEY.
Secrets are never displayed here. Monitoring is read-only: no wallet connection, private keys, custody, or trading.
