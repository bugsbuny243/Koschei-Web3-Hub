Core Tools UNIQUE

Every core user workflow appears once. Legacy Pro/Beta variants are kept inside the tool page instead of duplicated as separate dashboard cards.

Wallet Score
Get an instant reputation score for any Solana wallet based on public on-chain activity and trust signals.
POST /api/wallet/score
Token Scanner
Analyze any Solana token for authority, holder concentration, supply, liquidity and rug-pull risk signals.
POST /api/token/scan
Portfolio Tracker
Track SOL and token balances across multiple wallets in one unified dashboard.
POST /api/portfolio/track
Smart Money
Review smart-money style public signals and market context without wallet custody.
GET /api/smart-money
Watchlist
Monitor public wallets or contracts with Alchemy API polling and event summaries. This is the single Watchlist card.
POST /api/web3/sources
Risk Scanner
Run the core on-chain risk scanner for wallets, tokens, projects or public interaction targets.
POST /api/risk/scan
TX Decoder
Decode Solana transactions into clear summaries and preliminary risk hints. Pro/Beta depth stays inside this one tool page.
POST /api/tx/decode
