Koschei Agent API

Build autonomous agents and advanced workflows on Solana with authenticated Pro API access.

Agent API Overview

Authenticated endpoints for wallet scoring, risk analysis, metadata generation, and on-chain automation. Requires API key.

Getting Started

1. Get your Pro API key from the dashboard.
2. Use x-koschei-agent-key header for all requests.
3. All endpoints return structured JSON with risk signals and recommendations.

Core Endpoints

  • POST /api/agent/wallet/score — Advanced wallet reputation
  • POST /api/agent/risk/summary — Risk engine analysis
  • POST /api/agent/metadata/generate — AI metadata studio
  • GET /api/agent/health — Agent system health
