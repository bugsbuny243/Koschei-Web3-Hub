# Deployment

## Platform
- Railway for application hosting
- Neon Postgres for persistence

## Build/Start
Use existing project commands as-is:
- build: `next build`
- start: `next start`

## Scope Guardrails
Deployment is for no-custody Web Game Factory + Web3 Bridge only.
No wallet custody, minting, transaction signing, or contract deployment is included.

## RPC
Alchemy is read-only/future RPC support for chain data compatibility.
